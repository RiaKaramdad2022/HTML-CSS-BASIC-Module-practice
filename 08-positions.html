Positioning

The position property specifies the type of positioning method used for an element.

The four position values are:

  • fixed: this will pin something to a spot and it will stay there as the user scrolls. (ex.nav bars) Fixed position takes elements out of the normal document flow.
  • static: this is the default setting. It is not affected by top, bottom, left, and right properties. It is always positioned according to the normal flow of the page.
  • relative: does not take the element out of the normal doucment flow. Stil appear within its space/div/container, but allows it to be offset a bit from the top, bottom, right, and left. These properties will move it away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
  • absolute: takes element out of the normal doucment flow. If a parent element position is set to relative, and the child element is set to absolute, the child element's new (0,0) coordinate will be the upper left hand corner of the parent element instead of the whole page.

The z-index property allows you to overide the normal starcking order of the page. The default z-index is 0, adn you can set that value to be positive or negative. An element with a greater z-index will stack in front of an element with the lower z-index. If the two elements have the same z-index, or don't have one assigned, the element positioned last in the HTML will show up on top. You must have a position assigned in order to assign a z-index.
